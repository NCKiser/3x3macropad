This repository holds the QMK code and case STL for the 3x3 macropad.

Download and install Vial to re-configure your 3x3 macropad on the fly: https://get.vial.today/download/

Default Keymap is:\
[1][2][3]\
[4][5][6]\
[7][8][9/Fn]

and while holding [9/Fn]:\
[Esc][+][-]\
[Ret][*][/]\
[0][.] [trns]
